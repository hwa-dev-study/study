# Proxy 서버란
- 시스템에 방화벽을 가지고 있는 경우 외부와의 통신을 위해 만들어 놓은 서버이다.
- 클라이언트가 자신을 통해서 다른 네트워크 서비스에 간접적으로 접속할 수 있게 중계해주는 서버이다.
- <b>프록시 서버는 프록시 서버에 요청된 내용들을 캐시에 이용하여 저장한다.</b>

### Proxy 서버 사용 목적
- 프록시서버에 요청된 내용을 캐시를 사용해 저장해두며, 캐시에 저장된 내용을 재요청하는 경우 저장된 내용을 그대로
  돌려주면 되기때문에 전송 시간을 절약할 수 있다.
- 프록시 서버가 중간에 경유되게 되면 IP를 숨기는 것이 가능하여 보안 목적으로도 사용된다.
- 프록시 서버를 사용하면 저복을 다른나라로 우회할 수 있게 되어 접속 우회로도 사용된다.
